Capstone Project - The Battle of Neighborhoods

 

1.      Introduction/Business Problem

The problem is about if it's possible to compare neighborhoods of 5 multicultural cities (London, Sydney, Paris, Amsterdan and San Francisco) in order to find groups of neighborhoods that are similar according to similarities in the categories of businesses that are in them. This information can be used for example for travelers who wish to stay near a commercial area similar to where they come from, it could be also usefull for businesses that want to establish themselves in other cities of the world where there are commercial areas similar to the area where they currently are.

 

2.      Data

The data used for resolve the problem is:

-          Foursquare dataset: Foursquare is a local search-and-discovery service mobile app which provides search results for its users. The app provides personalized recommendations of places to go to near a user's current location based on users' previous browsing history, purchases, or check-in history. The Foursquare API allows application developers to interact with the Foursquare platform. The API itself is a RESTful set of addresses to which you can send requests, so there's really nothing to download onto your server.
